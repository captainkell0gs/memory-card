# Rick & Morty — Memory Card

A React memory game where you must click each character only once. Clicking the same card twice resets the round. Characters are fetched randomly from an external API so every round is different.

## Live Demo

https://rickandmorty-memory-alpha.vercel.app/

## Features

* Random characters fetched from API
* Cards shuffle after every click
* Score tracking
* Best score tracking
* Win detection when all cards are clicked
* Win popup modal
* Replay with new randomized cards
* Responsive layout

## Built With

* React
* JavaScript (ES6+)
* CSS
* Rick & Morty API
